# Polar Trip - gerador de formulário
Projeto de realizado no início do curso de desenvolvimento web no Instituto Infnet. O projeto foi desenvolvido como avaliação para o modulo de introdução ao desenvolvimento para web com uso de Node Js e o Express.js. O objetivo do mesmo foi tratar requisições utilizando as tecnologias mencionadas. Para o projeto foi feito como um MVC que tratava requisições de um formulário enviadas pelo usuário e retornava um PDF com as informações editadas no mesmo. 

## Linguagem Utilizada:
  - JavaScript

## Tecnologias e Bibliotecas Utilizadas:
  - Node.Js
  - HTML5
  - CSS3
  - Ejs
  - Express.Js
  - html-pdf-node
  - sass
  - nodemon (dev-dependencies)
    
## Requerimento:
  - NodeJS

## Primeiros Passos:
1. Faça a Instalação do NodeJS correspondente ao seu sistema operacional (caso ainda não esteja instalado). [Link de acesso a página](https://nodejs.org/en/).
2. Clone o projeto em uma pasta que deseja armazena-lo.
3. Na pasta raiz do projeto, instale as dependências com o comando <b>`npm install`</b> ou <b>`yarn install`</b> para instalar usando o yarn.
